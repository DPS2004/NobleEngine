new(StartingScene[, __transitionDuration[, __transitionType=Noble.Transition.CROSS_DISSOLVE[, __checkDebugBonks=false]]])
Engine initialization. Run this once in your main.lua file to begin your game.

Parameters:

  • StartingScene NobleScene This is the scene your game begins with, a title screen, loading screen, splash screen, etc. Pass the scene's class name, not an instance of the scene.
  • __transitionDuration number If you want to transition from the final frame of your launch image sequence, enter a duration in seconds here. (optional)
  • __transitionType Noble.TransitionType If a transition duration is set, use this transition type. (default Noble.Transition.CROSS_DISSOLVE)
  • __checkDebugBonks bool Noble Engine-specific errors are called "bonks." Set this to true during development to check for more of them. It is resource intensive, so turn it off for release. (default false)

transition(NewScene[, __duration=1[, __transitionType=Noble.TransitionType.DIP_TO_BLACK[, __holdDuration=0.2]]])
Transition to a new scene. This method will create a new scene, mark the previous one for garbage collection, and animate between them.

Parameters:

  • NewScene NobleScene The scene to transition to. Pass the scene's class name, not an instance of the scene. You always transition from Noble.currentScene
  • __duration number The length of the transition, in seconds. (default 1)
  • __transitionType Noble.TransitionType If a transition duration is set, use this transition type. (default Noble.TransitionType.DIP_TO_BLACK)
  • __holdDuration number For DIP transitions, the time spent holding at the transition midpoint. Does not increase the total transition duration, but is taken from it. So, don't make it longer than the transition duration. (default 0.2)
