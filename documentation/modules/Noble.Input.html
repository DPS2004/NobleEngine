Functions

Noble.Input.getHandler()
Get the currently active input handler. Returns nil if none are active.

Returns:

    table A table of callbacks which handle input events.
Noble.Input.setHandler([__inputHandler=nil])
Use this to change the active inputHandler. Enter nil to disable input.

Parameters:

  • __inputHandler table A table of callbacks which handle input events. (default nil)

See also:

    NobleScene.inputHandler
Noble.Input.disable()
A helper function that calls Noble.Input.setHandler() with no argument.

See also:

    setHandler
Noble.Input.activateCrankIndicator(__bool)
Enable/disable on-screen crank indicator.

NOTE: The indicator will only ever show if the crank is docked.

Parameters:

  • __bool Set true to start showing the on-screen crank indicator. Set false to stop showing it.
Noble.Input.crankIndicatorActive()
Checks whethe the crank indicator has been activated via Noble.Input.showCrankIndicator(true).

NOTE: The indicator will only ever show if the crank is docked.
