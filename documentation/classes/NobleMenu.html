Methods

NobleMenu:new(__menuItems, __localized, __color, __padding, __horizontalPadding, __font, __crankControlled, __dPadControlled, __selectedOutlineThickness)
Create a new menu object

Parameters:

  • __menuItems This is a table of strings, either menu item names, or localization keys.
  • __localized Set to true to localize __menuItems. Default: false
  • __color Graphics.kColorBlack or Graphics.kColorWhite. Default: Graphics.kColorBlack
  • __padding Cell padding for menu items.
  • __horizontalPadding Use this to override horizontal padding, useful for certain fonts. If nil, uses __padding.
  • __font If nil, uses current set font.
  • __crankControlled any
  • __dPadControlled any
  • __selectedOutlineThickness Sets the outline thickness for selected items. Default: 2

Returns:

    A new menu
